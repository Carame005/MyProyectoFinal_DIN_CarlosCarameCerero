# Manual de Usuario — AutiCare (versión básica)

Este manual está pensado para personas con Alzheimer y para sus cuidadores. Explica de forma clara y paso a paso cómo usar la aplicación, con lenguaje sencillo y recordatorios prácticos.

Introducción rápida
-------------------
- Qué hace la app: ayuda a organizar tareas, eventos y colecciones de vídeo. Permite programar recordatorios y ver su información de forma clara.
- Objetivo: facilitar la rutina diaria y conservar recuerdos en vídeo.
- Público: personas con dificultades de memoria y sus cuidadores.

Conceptos básicos (pantallas principales)
-----------------------------------------
- Inicio (Inicio/Resumen): muestra el saludo, la fecha y tarjetas con conteo de tareas, eventos y colecciones.
- Tareas: lista las tareas personales. Puede marcar una tarea como en progreso o hecha.
- Calendario: ver y añadir eventos en días concretos; puede añadir una hora para recibir un recordatorio.
- Progreso / Vídeos: colecciones de vídeos que agrupan recuerdos.
- Ajustes: menú lateral con opciones (cambiar nombre, tema, notificaciones) y opción para salir.

Primeros pasos (configurar cuenta)
----------------------------------
1. Abrir la app: en la pantalla de inicio verá un formulario de inicio de sesión o de registro.
2. Registrar una cuenta (si no tiene): escoger "Registrarse" y completar:
   - Nombre (cómo desea que le llamen)
   - Correo electrónico (una dirección válida)
   - Contraseña (algo fácil de recordar para usted y el cuidador)
   - Pulse "Registrar" para crear la cuenta.
3. Si ya tiene cuenta: elija "Iniciar sesión", escriba su correo/usuario y la contraseña y pulse "Entrar".

Consejo para cuidadores: al crear la cuenta, anote la contraseña en un lugar seguro o configure una contraseña sencilla y segura para la persona.

Cómo iniciar sesión
-------------------
1. En la pantalla de inicio, introduzca su usuario o correo en el campo superior.
2. Escriba su contraseña en el campo de contraseña.
3. Pulse "Entrar".
4. Si olvida la contraseña, el cuidador puede registrar un nuevo usuario o purgar datos de prueba (opción de purga está para pruebas; usar con precaución).

Cómo ver la pantalla principal
-----------------------------
- Después de iniciar sesión verá:
  - Su nombre arriba (bien visible).
  - La fecha del día.
  - Tres tarjetas: Tareas, Eventos, Colecciones. Cada una muestra el número correspondiente.
- Toque una tarjeta para ver más detalles.

Tareas (añadir, marcar, eliminar)
---------------------------------
- Ver tareas:
  - Pulse la tarjeta "Tareas" o vaya a la pestaña Tareas en la barra inferior.
- Añadir una tarea:
  1. Pulse el botón circular azul con el símbolo "+" en la parte inferior.
  2. Aparecerá un formulario. Escriba el título y descripción (opcional).
  3. Pulse "Añadir".
- Cambiar el estado de una tarea:
  - Toque la tarea para desplegar opciones.
  - Pulse "En progreso" o "Hecho" según corresponda.
- Eliminar una tarea:
  - Si la tarea está marcada como "Hecho", verá un icono de papelera; púlselo para eliminar.

Consejo: mantener las tareas cortas y concretas (por ejemplo: "Tomar medicación a las 9:00").

Calendario y eventos (añadir recordatorio)
------------------------------------------
- Abrir calendario: pulse la pestaña "Calendario" en la barra inferior.
- Navegar por meses: use las flechas izquierda/derecha para cambiar el mes.
- Seleccionar día: pulse el día en el calendario.
- Ver eventos del día: en la lista inferior aparecen los eventos del día seleccionado.
- Añadir evento y recordatorio:
  1. Pulse el botón azul "+" en la parte inferior.
  2. Escriba el título del evento.
  3. (Opcional) Pulse "Seleccionar hora" para escoger una hora (si quiere un recordatorio a una hora concreta).
  4. Pulse "Guardar".
- Eliminar evento:
  - En la lista de eventos pulse el icono de papelera junto al evento.

Aviso sobre notificaciones: Para recibir recordatorios debe permitir las notificaciones en el dispositivo. Si no llegan notificaciones, revise los ajustes del dispositivo.

Colecciones de vídeos (añadir y reproducir)
-------------------------------------------
- Acceder: vaya a la pestaña "Progreso/ Vídeos" (o "Stats").
- Añadir una colección:
  1. Pulse el botón circular azul con "+".
  2. Escriba un título para la colección y pulse "Crear".
- Añadir un vídeo a una colección:
  1. Abra la colección (pulse su tarjeta para expandirla).
  2. Pulse el icono de añadir vídeo dentro de la colección.
  3. Seleccione "Seleccionar vídeo" y elija un archivo de la galería o archivos del dispositivo.
  4. Rellene título y descripción si lo desea y pulse "Añadir".
- Reproducir un vídeo:
  - Pulse el icono de reproducir junto al vídeo. Se abrirá un reproductor que permite ampliar a pantalla completa y cerrar.
- Eliminar vídeo o colecciones: use el icono de papelera junto al elemento que desea eliminar.

Perfil y ajustes (cambiar nombre, foto y cerrar sesión)
--------------------------------------------------------
- Abrir menú de perfil: pulse el avatar (foto) en la parte superior izquierda.
- Cambiar foto: en el menú de perfil pulse la foto y seleccione una nueva de la galería.
- Ajustes: pulse el icono de ajustes en el topbar para abrir el panel lateral con opciones:
  - Cambiar nombre: escriba el nuevo nombre y pulse "Guardar".
  - Cambiar contraseña: rellene la contraseña actual y la nueva (si procede).
  - Filtro claro (tema): cambiar formato visual para facilitar la lectura.
  - Notificaciones: activar/desactivar.
  - Función tutor (opción para usuarios tutorizados): en el panel lateral aparece un interruptor "Función tutor" que permite al usuario aceptar o rechazar ser seleccionable como tutorizado por un tutor.
    - Estado por defecto: activado (el usuario puede ser elegido como tutorizado por un tutor).
    - Restricción importante: si el usuario ya ha sido seleccionado como tutorizado por algún tutor, intentar desactivar esta opción mostrará un mensaje de error indicando que debe pedir al tutor que lo elimine de su lista antes de desactivar la función. Esto evita inconsistencias entre la configuración del usuario y la lista del tutor.
    - Cambio persistente: al activar/desactivar se guarda la preferencia en la configuración del usuario.
  - Cerrar sesión: volver a la pantalla de inicio de sesión.

Nota sobre gestión de tutorizados y pantallas relacionadas
-----------------------------------------------------
- Gestión centralizada: la lista y gestión de tutorizados debe realizarse desde la pantalla de Tutor (o desde la sección dedicada en la barra inferior para tutores), donde un tutor puede ver sus tutorizados y añadir/quitar tareas o eventos para ellos. El interruptor "Función tutor" en Ajustes es una preferencia personal para aceptar ser tutorizado; no sustituye la gestión por parte de un tutor.
- Mensajes y errores: si al intentar desactivar la "Función tutor" aparece un mensaje indicando que no puede desactivarlo, significa que existe al menos un tutor que actualmente tiene al usuario en su lista; en ese caso el usuario debe solicitar al tutor que lo elimine previamente.

Notificaciones y recordatorios
------------------------------
- Para que los recordatorios funcionen debe aceptar notificaciones en el dispositivo.
- Si no recibe notificaciones:
  1. Compruebe en los Ajustes del teléfono que las notificaciones de la app están activadas.
  2. Compruebe que la hora y fecha del dispositivo son correctas.

Buenas prácticas para cuidadores
-------------------------------
- Mantener la cuenta y contraseña en un lugar seguro y accesible para el cuidador.
- Configurar un correo electrónico o contacto de confianza para recuperar acceso si fuera necesario.
- Revisar periódicamente que los recordatorios están activos y que el usuario reconoce las notificaciones.
- Enseñar al usuario a utilizar la app con ejercicios simples (por ejemplo: crear una tarea diaria juntos).

Preguntas frecuentes (solución de problemas)
-------------------------------------------
- No puedo iniciar sesión:
  - Verifique que el correo/usuario y la contraseña estén correctos.
  - Si no funciona, el cuidador puede crear un nuevo usuario o usar la opción de purga (si está disponible) con precaución.

- No aparecen mis vídeos después de seleccionarlos:
  - Asegúrese de conceder permisos para acceder a archivos/galería.
  - Algunos URIs pueden requerir permisos persistentes; intente volver a seleccionar el vídeo.

- No recibo recordatorios:
  - Asegúrese de que las notificaciones están activadas para la app en Ajustes del dispositivo.
  - Compruebe que el evento tiene hora y que la hora no está en el pasado.

Seguridad y privacidad
----------------------
- Mantenga el dispositivo en un lugar seguro.
- No comparta su contraseña con personas no de confianza.
- El cuidador puede revisar y editar datos si la persona usuaria no puede hacerlo sola.

Glosario (palabras simples)
- Colección: un grupo de vídeos guardados con un título.
- Evento: algo programado en el calendario (puede tener hora para recordatorio).
- Tarea: acción que se debe realizar (marcar cuando está hecha).

Contacto y soporte
------------------
- Si necesita ayuda técnica contacte con la persona responsable de soporte de la app (anotar contacto localmente).

Notas finales para cuidadores
-----------------------------
- Sea paciente al enseñar la app: repita pasos simples y supervise las primeras veces.
- Ajuste la app al usuario: textos grandes, contraste alto y alertas son útiles.
