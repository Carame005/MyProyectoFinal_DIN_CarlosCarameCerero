# Documentación del proyecto

Esta carpeta contiene la documentación técnica y de usuario del proyecto "MyProyectoFinal_DIN_CarlosCarameCerero" (módulo app).

Archivos incluidos:

- `ARCHITECTURE.md` : Descripción de la arquitectura, módulos y decisiones principales.
- `LIBRARIES.md` : Lista de librerías y razones de uso.
- `COMPONENTS.md` : Catálogo de componentes Compose reutilizables y su API.
- `PERSISTENCE.md` : Formato y ubicación de datos persistentes (ficheros/SharedPreferences).
- `SECURITY.md` : Riesgos detectados y recomendaciones (credenciales, permisos).
- `DEV_SETUP.md` : Cómo preparar el entorno para desarrollar y comandos útiles (PowerShell).
- `TESTS.md` : Estrategia de pruebas y qué tests existen actualmente.
- `TODO.md` : Que falta por mejorar y siguientes pasos recomendados.

Uso rápido
---------
Abrir estos archivos desde el IDE para conocer cómo está organizado el proyecto, cómo se persisten los datos, los puntos críticos de seguridad y las recomendaciones para avanzar.

Si quieres que genere versiones más detalladas (por ejemplo un manual de usuario PDF o tutoriales paso a paso), dímelo y lo preparo en la misma carpeta.
